# Srini Personal FinanceAI - Project Overview

## 🎯 Vision

An AI-powered personal finance assistant that aggregates income data from multiple financial sources, analyzes mortgages, credit cards, and debts, and provides actionable recommendations to help users make optimal financial decisions.

## 📋 Project Scope

### Core Problem Solved
- **Information Fragmentation**: Financial data scattered across multiple banks, cards, and platforms
- **Decision Paralysis**: Difficulty determining best actions (refinance? payoff strategy? card optimization?)
- **Analysis Burden**: Complex calculations require specialized knowledge

### Target Users
- Homeowners considering mortgage refinancing
- Consumers with multiple credit cards wanting optimization
- Individuals managing multiple debts
- Anyone seeking automated financial planning

## 🚀 Quick Start

```bash
# Clone repository
git clone <repository-url>
cd personal-finance-assistant

# Install dependencies
npm install

# Setup environment
cp .env.example .env

# Run development server
npm run dev
```
